## ✨ Features

### 🎨 Design & UI
- **Dark Theme** — Professional dark color scheme with cyan and purple accents
- **Responsive Design** — Fully responsive across all devices (Desktop, Tablet, Mobile)
- **Glassmorphism Effects** — Modern glass-like UI components with backdrop blur
- **Gradient Accents** — Beautiful gradient text, buttons, and decorative elements
- **Smooth Animations** — CSS and JavaScript-powered scroll reveal animations
- **Particle Background** — Interactive canvas-based particle animation system
- **Custom Scrollbar** — Themed scrollbar matching the overall design

### ⚡ Technical Features
- **No External JS Frameworks** — Built with vanilla JavaScript for optimal performance
- **CSS Custom Properties** — Centralized theming with CSS variables
- **Intersection Observer API** — Efficient scroll-based animations
- **Canvas Particle System** — Custom particle animation with connection lines
- **Typing Effect** — Multi-phrase typing animation for hero section
- **Form Handling** — Server-side form processing via FormSubmit.co
- **Lazy Loading** — Deferred image loading for faster page loads
- **SEO Optimized** — Meta tags, Open Graph, and semantic HTML
- **Accessibility** — ARIA labels, keyboard navigation, and screen reader support

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- [Git](https://git-scm.com/) installed on your machine

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/pmanojan/pmanojan.github.io.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd pmanojan.github.io
   ```

3. **Open in browser**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

4. **Visit the site**
   ```
   http://localhost:8000
   ```

### Deployment

The site is deployed using **GitHub Pages**. Any push to the `main` branch automatically updates the live site.

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

## 🔧 Customization Guide

### Changing Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary: #00d4ff;      /* Change primary color */
    --secondary: #7c3aed;    /* Change secondary color */
    --bg-primary: #0a0a0f;   /* Change background */
}
```

### Adding New Projects
Add project data in `script.js`:
```javascript
const projectsData = {
    'new-project': {
        title: 'My New Project',
        description: 'Project description...',
        icon: 'fa-code',
        category: 'Development',
        techStack: ['Tech1', 'Tech2'],
        githubUrl: 'https://github.com/...'
    }
};
```

### Updating Contact Form
Change the FormSubmit email in `index.html`:
```html
<form action="https://formsubmit.co/YOUR_EMAIL@gmail.com" method="POST">
```
