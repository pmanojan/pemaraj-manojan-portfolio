## ✨ Features

### 🎨 Design & UI
- **Dark Theme** — Professional dark color scheme with cyan and purple accents
- **Responsive Design** — Fully responsive across all devices (Desktop, Tablet, Mobile)
- **Glassmorphism Effects** — Modern glass-like UI components with backdrop blur
- **Gradient Accents** — Beautiful gradient text, buttons, and decorative elements
- **Smooth Animations** — CSS and JavaScript-powered scroll reveal animations
- **Particle Background** — Interactive canvas-based particle animation system
- **Custom Scrollbar** — Themed scrollbar matching the overall design

### 📄 Sections
| Section | Description |
|---------|-------------|
| 🏠 **Hero** | Animated introduction with typing effect, floating cards, and stats counter |
| 👤 **About** | Personal bio with profile image, details, and downloadable CV |
| 🛠️ **Skills** | Categorized skill bars with animated progress indicators |
| 💼 **Experience** | Interactive timeline showcasing professional journey |
| 📂 **Projects** | Filterable project cards with modals and GitHub links |
| 🎓 **Education** | Academic background with badges and descriptions |
| 📜 **Certifications** | Filterable certification cards (Cisco, Security, Others) |
| 📬 **Contact** | Contact form with FormSubmit integration and social links |
| 🙏 **Thank You** | Animated confirmation page with auto-redirect countdown |

### ⚡ Technical Features
- **No External JS Frameworks** — Built with vanilla JavaScript for optimal performance
- **CSS Custom Properties** — Centralized theming with CSS variables
- **Intersection Observer API** — Efficient scroll-based animations
- **Canvas Particle System** — Custom particle animation with connection lines
- **Typing Effect** — Multi-phrase typing animation for hero section
- **Form Handling** — Server-side form processing via FormSubmit.co
- **Lazy Loading** — Deferred image loading for faster page loads
- **SEO Optimized** — Meta tags, Open Graph, and semantic HTML
- **Accessibility** — ARIA labels, keyboard navigation, and screen reader support
