# 🌐 Portfolio Website - Network Administrator & Cybersecurity Specialist

A modern, professional portfolio website showcasing expertise in Network Administration, 
Cybersecurity, and IT Education. Built with pure HTML, CSS, and JavaScript.

## ✨ Features

- 🎨 **Animated Particle Background** - Interactive network visualization
- ⌨️ **Dynamic Typing Effect** - Multiple role showcase
- 📊 **Skill Progress Bars** - Visual skill representation
- 🎓 **Certification Showcase** - Filterable certification gallery
- 📱 **Fully Responsive** - Mobile-first design
- 🌙 **Dark Cyber Theme** - Professional cybersecurity aesthetics
- 📧 **Contact Form** - Integrated with FormSubmit
- 🔄 **Smooth Animations** - Scroll-triggered reveal effects

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, JetBrains Mono)

## 🚀 Quick Start

1. Clone the repository
2. Replace `image.jpg` with your photo
3. Update personal information in HTML
4. Deploy to GitHub Pages or any hosting service
